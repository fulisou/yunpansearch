# yunpansearch
云盘搜索（yunpansearch）是一套个人网盘资源管理程序,可以帮你管理、搜索自己的网盘资源

云盘搜索支持百度网盘，阿里云盘，夸克网盘，UC网盘，迅雷网盘五大网盘搜索

Java8 + springboot + maven + ES +前端 <br>

需要源码加微信：senemhi 备注：网盘源码。 程序永久使用，并包 1 年更新，可以帮忙安装（服务器、域名提前解析好）现有数据可打包
现有数据量100万+
